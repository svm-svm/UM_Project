# EduPro Demand & Revenue Forecasting

## Overview
This project builds machine learning models to predict course enrollments and revenue for an online learning platform (EduPro).

## Features
- Course demand prediction
- Revenue forecasting
- Feature importance analysis
- Streamlit dashboard

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run data preparation

python src/data_prep.py

### 3. Train model

python src/model.py

### 4. Run dashboard

streamlit run src/course_dashboard.py

## Output
- Predicts enrollments and revenue
- Interactive dashboard for decision making
